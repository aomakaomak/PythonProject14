# Финансовый анализатор

Учебный проект на Python для анализа банковских операций из Excel-файла.  
Позволяет формировать отчёты по тратам, кэшбэку, валютам и акциям.  
Результаты выдаются в формате JSON или словарей для удобства интеграции.

---

## 🚀 Возможности

- Чтение и предобработка Excel-файлов с операциями.
- Расчёт кэшбэка по картам.
- Определение **топ-5 операций** по сумме.
- Получение курсов валют через API.
- Получение стоимости акций через API.
- Формирование агрегированного JSON-ответа (`main`).
- Анализ кэшбэка по категориям за конкретный месяц.
- Подсчёт трат по категории за последние 3 месяца.
- Основная точка входа `main_func`, объединяющая все сервисы.

🔑 Основные функции
utils.py

read_file(file: str) — чтение Excel, очистка номеров карт.

top_5_operations(df) — топ-5 операций по сумме.

api_currency(currency: str) — курс валюты через API.

currency_rate(file: str) — словарь валют и их курсов.

api_stock(stock: str) — цена акции через API.

stock_rate(file: str) — словарь акций и их цен.

views.py

main(date_time: str) — сбор JSON-ответа из всех сервисов.

services.py

benefit_categories(data, year, month) — кэшбэк по категориям за месяц.

reports.py

last_3_months_operations(data, category, ref_date) — траты по категории за 3 месяца.

main.py

main_func() — главная функция, печатает все отчёты и возвращает "Спасибо за ваш запрос".
